Introducción a HTML

En su esencia, HTML es un lenguaje compuesto por elementos que se pueden aplicar a fragmentos de texto para otorgarles un significado diferente en un documento (¿Es un párrafo? ¿Es una lista con viñetas? ¿Forma parte de una tabla?), estructurar un documento en secciones lógicas (¿Tiene encabezado? ¿Tres columnas de contenido? ¿Un menú de navegación?) e incrustar contenido, como imágenes y vídeos, en una página. Este módulo presentará los dos primeros aspectos mencionados, así como los conceptos fundamentales y la sintaxis que debes conocer para comprender HTML. 

¿Interesado en convertirte en desarrollador web de front-end? Hemos creado un curso que incluye toda la información esencial que necesitas para trabajar en pro de tu objetivo.

Requisitos previos

Antes de comenzar este módulo, no necesitas tener conocimientos previos de HTML, pero debes estar familiarizado al menos en nivel básico con el uso de computadoras y el uso pasivo de la web (es decir, solo mirarla y consumir contenido). Debes tener configurado un entorno de trabajo básico (como se detalla en la instalación de software básico) y entender cómo crear y gestionar archivos (como se detalla en la manipulación de archivos). Ambos aspectos forman parte de nuestro módulo para principiantes completo, "Introducción al mundo de la web".

Aplicaciones de HTML

Como se mencionó anteriormente, HTML es uno de los lenguajes más utilizados en la web. A continuación, enumeraré algunos de ellos:

  • Desarrollo de páginas web:
    HTML se utiliza para crear páginas que se muestran en la web. Casi todas las páginas web contienen etiquetas de HTML para mostrar su contenido en el navegador.
  • Navegación en Internet:
    HTML proporciona etiquetas que se utilizan para navegar de una página a otra y se utiliza ampliamente en la navegación por Internet.
  • Interfaz de usuario adaptable:
    Las páginas de HTML funcionan bien en todas las plataformas, como dispositivos móviles, tabletas, computadoras de escritorio o portátiles, gracias a las estrategias de diseño adaptable.
  • Soporte sin conexión:
    Una vez cargadas, las páginas de HTML pueden estar disponibles sin conexión en la máquina sin necesidad de Internet.
  • Desarrollo de juegos:
    HTML5 cuenta con soporte nativo para ofrecer una experiencia rica y ahora también es útil en el ámbito del desarrollo de juegos.

Público objetivo

Este tutorial de HTML está diseñado para los aspirantes a diseñadores y desarrolladores web que necesitan comprender HTML con suficiente detalle, además de tener una visión general y ejemplos prácticos. Este tutorial te proporcionará los elementos suficientes para comenzar con HTML y alcanzar un nivel de experiencia más avanzado.
